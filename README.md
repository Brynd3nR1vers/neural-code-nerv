# 🌐 NERV_DATABASE_SYSTEM // UNIT_01
### [ REPOSITORIO DE TECNOLOGÍA TÁCTICA Y SUMINISTROS IA ]

Este proyecto es una plataforma de E-Commerce de alto rendimiento diseñada bajo los protocolos estéticos de **Neon Genesis Evangelion**. Especializada en la distribución de software de sincronización LCL y componentes de enlace neural para pilotos.

---

## 🛠️ ESPECIFICACIONES TÉCNICAS
El sistema ha sido construido utilizando el stack de desarrollo **C-01**:

* **Frontend:** React.js con Vite.
* **Routing:** React Router Dom (Navegación dinámica por sectores).
* **Database:** Firebase Firestore (Sincronización de registros en tiempo real).
* **State Management:** Context API (Gestión de manifiesto de carga / Carrito).
* **Estilos:** CSS3 nativo con arquitectura de "High-Density UI".

---

## 🔋 CARACTERÍSTICAS DEL SISTEMA

### 1. INTERFAZ TÁCTICA (UI/UX)
- **Home Terminal:** Pantalla de inicio interactiva con imágenes cyberpunk y explicación de sectores.
- **Protocolo de Acceso:** Botón de ingreso con simulación de carga (Loading Screen) para validación de credenciales NERV.
- **Diseño Responsivo:** Menú tipo hamburguesa optimizado para dispositivos móviles tácticos.

### 2. NÚCLEO DE DATOS (Firebase)
- Sincronización automática de productos desde la nube.
- Sistema de filtrado inteligente por categorías (`Software` / `Neural`).
- Protocolo de rescate: Los ítems sin categoría definida se mantienen visibles bajo estado de emergencia.

### 3. MANIFIESTO DE CARGA (Carrito)
- Persistencia de datos durante la sesión.
- Cálculo de totales con corrección de errores numéricos (Fix NaN).
- Validación de salida en el sector de Checkout.
